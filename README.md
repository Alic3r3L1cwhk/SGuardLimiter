# SGuardLimiter
这是一个ACE反作弊程序一键限制脚本，原理基于华硕官方发布的视频。

[[补档2] 关闭ACE反作弊系统 保护你的电脑性能](https://www.bilibili.com/video/BV1ca1WBUE8j/?spm_id_from=333.337.search-card.all.click)

使用方法:以管理员方式运行ACE限制.exe

或者编译源代码后发布，以管理员身份运行\SGuardLimiter\bin\Release\net8.0\win-x64\publish\SGuardLimiter.exe。
